# ABYSS DOMINION Canvas v0.4

Canvasで作った戦闘コア検証版。

## 見るポイント
- 魔物が歩く動きは気持ちいいか
- 敵とぶつかった瞬間が分かりやすいか
- ダメージ数字・DPS・HP変化が気持ちいいか
- スキル演出だけで押したくなるか
- 勝利後にもう1階進みたくなるか

- Canvas 60FPS描画
- 自動歩行
- 自動戦闘
- リアルダメージ数字
- 残HP表示
- DPS表示
- 4スキル
- パーティクル
- 画面シェイク
- ゴールド吸収演出
- 報酬3択

- 1000階

# ABYSS DOMINION α v0.6

ポケットモンスターの洞窟探索に近い、見下ろし型探索α版。

## 主な変更
- 19×19のランダム洞窟
- タップした地点まで自動移動
- カメラ追従
- 右上ミニマップ
- 未探索部分は非表示
- 勇者が洞窟内を巡回
- 勇者に見つかると追跡される
- 宝箱2個
- 階段
- 勇者を倒した後に階段へ行くと階層クリア
- 戦闘は同じ洞窟画面内で発生
- 魔物4体が個別攻撃
- リアルタイムダメージ・クリティカル・DPS
- 戦闘後の個別戦績とMVP
- 報酬3択を完全固定
- 報酬は1回だけ選択可能
- 強化量を2〜3%程度へ抑制
- リアルタイム保存

GitHub Pagesでは index.html / style.css / script.js をリポジトリ直下へ配置してください。

## 主な実装

- タップ移動、1本指パン、2本指ズーム
- マップ外タップ無効、カメラ範囲制限
- ランダムダンジョン、敵巡回・追跡・画面外補充
- 50階ごとのエリアテーマ
- 3〜7階間隔でランダム出現する安全なセレクトショップ
- ベッド全回復、商人の購入・売却・装備
- 専用ターン制戦闘、AUTO/MANUAL、速度変更
- MANUAL中は敵味方とも完全停止
- 行動順、現在ターン強調、攻撃モーション、ダメージ演出
- 通常攻撃、ガード、複数スキル、アイテム、捕獲、逃走
- バフ・デバフ表示、全滅処理
- 経験値、Lv999、レベルアップ演出、重複+値、魔魂合成
- 宝箱の多種ドロップ、装備レアリティ、ランダム能力
- 武器・防具・アクセサリー
- 1連・10連召喚
- 10階チェックポイント、階層ワープ
- ローカルリアルタイム保存、セーブ書き出し

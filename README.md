# ダンジョン経営 Ver.0.3

## 収録機能
- 8×8ランダムマップ、入口・魔王・岩の再生成
- リアルタイム勇者進行・魔物戦・魔王戦
- 魔物死亡、負傷、10分ごとの負傷回復
- デイリー、ノーマル、プレミアガチャ
- ステージ進行で排出率変化、ガチャ天井
- 部族解禁、重複素材、確率合成、失敗補正
- 部族熟練度、進化候補表示
- 部族単位の装備
- 勇者ドロップ、装備倉庫
- 勇者学習値による強化
- 図鑑・実績・ローカルセーブ

## 導入
既存リポジトリの `index.html`、`style.css`、`script.js` をこの3ファイルで置き換えてください。
古いセーブとは互換性がないため、新規ゲーム開始を推奨します。
